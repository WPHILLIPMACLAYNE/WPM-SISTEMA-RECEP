Como usar

Este arquivo funciona sozinho no navegador.

1. Abra o HTML no navegador
Pode ser Chrome, Edge ou outro navegador moderno.
2. Os dados ficam salvos localmente
Tudo é persistido em localStorage no seu navegador.
3. Faça backup quando quiser
Use Exportar backup para baixar um JSON com tudo.
4. Edite nomes e estrutura
Na aba Configurações você adapta o app ao seu processo.
